结构分类

流程控制的结构，可分为如下几类：

  • 顺序结构：程序从上往下依次执行。

  • 分支结构：从多条路径中选择一条去执行。

  • 循环结构：在满足一定条件的基础上，重复执行一段代码。

顺序结构

SQL语句默认就是从上往下依次执行。

分支结构

分支结构主要有以下几种语法：

  • IF函数

  • CASE结构

  • IF结构

1. IF函数

1

IF(表达式1，表达式2，表达式3)

如果表达式1成立，则IF函数返回表达式2的值，否则返回表达式3的值。

循环结构

循环结构主要有以下几种语法：

  • WHILE

  • LOOP

  • REPEAT

控制循环流程的语法如下：

  • ITERATE：类似于continue。

  • LEAVE：类似于break。

1. 循环结构语法

标签主要用于搭配iterate和leave跳出循环。

DELIMITER $
CREATE PROCEDURE test_while(n int)
BEGIN
    declare sum int default 0;
    WHILE n > 0 DO
        SET sum=sum+n;
        SET n=n-1;
    END WHILE;
    SELECT sum AS result;
END$

CREATE PROCEDURE test_repeat(n int)
BEGIN
    declare sum int default 0;
    REPEAT
        SET sum=sum+n;
        SET n=n-1;
    UNTIL n = 0
    END REPEAT;
    SELECT sum AS result;
END$

DEMILITER $
CREATE PROCEDURE test_loop(n int)
BEGIN
    declare sum int default 0;
    L: LOOP
        SET sum=sum+n;
        SET n=n-1;
        IF n=0 THEN
            leave L;
        END IF;
    END LOOP L;
    SELECT sum;
END$
